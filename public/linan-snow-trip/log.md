# 临安百丈岭看雪 - 素材记录

## 5:33 出发前
- 天还没亮，窗外漆黑
- 等老婆洗漱
- 目的地：临安百丈岭
- 原因：杭州难得遇上寒潮，朋友圈都在说百丈岭要下大雪
- 特意请了假

## 5:42
- 在热卤蛋，一共4个，前几天卤好的
- 作为早餐

## 5:45 左右
- 老婆提醒带保温毯
- 最近看了很多鳌太线的视频，要做足安全措施

## 5:57
- 还没出发
- 在等老婆化妆
- 还要等狗吃饭，狗叫七月
- 老婆还没吃早餐（卤蛋）
- 卤蛋热太久了，太软烂了，有点难吃
- 补充：不是老，是太软烂

## 6:06
- 老婆给我做了一碗麦片
- 麦片里加了很多不同种类的坚果
- 早餐挺丰盛的

## 写作工具
- 这篇文章用 Claude Code + Happy 服务写的
- Happy 服务：电脑本地起一个 Claude Code session，手机扫码连接
- 很方便，可以边走边口述记录

## 6:28
- 给狗（七月）穿好了衣服
- 准备出发，但还没出门

## 6:36
- 终于上车了
- 接下来有一个多小时的车程
- 正式出发去百丈岭

## 路上观察
- Happy 服务在国内移动网络下正常使用
- 疑问：是不是用了 P2P 直连模式，没经过服务器？
- 待研究

## 导航
- 目的地：上溪村（百丈岭附近）

## 6:48
- 室外温度：3℃
- 距离目的地：还有 1 小时 32 分

## Happy 体验反馈
- 问题：不支持上传图片
- 这点有些不方便，原因未知
- 问题2：需要先在电脑上起一个 Claude Code session，手机才能连接
- 没有找到从手机端直接创建 session 或指定目录的方法
- 比如想远程在某个目录下新开一个项目，好像做不到
- 不确定是没找到方法还是功能本身没做
- 待探索

## 6:50
- 还没上高速
- 到青山湖了

## 8:34
- 到达转山驿站附近
- 在上厕所
- 停的车挺多的
- 来晚的话停车会远，要走更长距离

## 入口
- 通过入口了
- 收卫生管理费，每人 10 块
- 雪挺厚的
- 听说今天上山的人很多，前面已经有几百人在爬了

## 9:03
- 已经爬了一段距离
- 身体发热，把羽绒服脱了
- 避免出汗弄湿衣服

## 9:07
- 身子热起来了
- 山里非常宁静
- 停下来能听到雪打在身上的声音
- 适合冥想

## 继续上山
- "银装素裹"这个小学用烂的词，在这里真的用得上
- 雪的颗粒度变大了
- 细雪掺杂着鹅毛大雪的感觉
- 近距离看结冰的森林，有种往外刺的感觉
- 好像有了生命力，像一个寒冰领主
- 头发结冰了，一串串的冰碴子
- 不想戴帽子，戴着太闷热
- 狗子七月很受欢迎
- 今天打扮得比较精神，好多人想和它合照

## 9:58
- 身体已经适应了徒步爬山
- 达到了动态平衡
- 不会特别累，但又在持续输出功率的感觉

## 10:26
- 已经走了一个半小时
- 山上风景特别美
- 林间小道配雪景，把人映得很白净
- 狗子七月很受路人欢迎
- 感受到很多善意
- 徒步的人比城里人对狗更友好

## 休息吃东西
- 在吃沙琪玛
- 发现低温下沙琪玛比平时脆很多，口感更好

## 路上的平台
- 发现一个平台，很多人聚集
- 有人在尬舞
- 有桌子凳子可以吃饭
- 户外也是一种社交

## 吐槽
- 山上居然有人边走边抽烟
- 是位女士，抽的女士烟
- 女士烟味道最臭
- 太没公德心了

## 11:02
- 快到中午了
- 雪停了
- 地上积雪开始融化
- 下山可能要穿冰爪

## 11:10
- 气喘吁吁
- 有一段连续很长的上坡阶梯
- 大腿有泵感，像骑动感单车那种感觉

## 接近山顶 (11:21)
- 不能停下来，停下来会非常冷
- 必须保持上升前进
- 快到剑湖了
- 在山顶的平路上走
- 山上白茫茫的，有点雾

## 11:56 到达剑湖
- 到达目的地剑湖
- 雾茫茫一片
- 湖面非常平静
- 有点像日本侘寂风，很安静很平静

## 剑湖遇野狗
- 在剑湖平台上遇到一只野狗
- 黑白相间的狗
- 一直粘着七月，想跟七月玩
- 但七月怕大狗，只好把七月抱起来
- 这只狗非常热情，一直跟着我们
- 我们走哪它就在前面走着
- 只能一直抱着七月，有点不方便

## 开始下山
- 剑湖看久了也没啥，白茫茫一片
- 从剑湖出发原路返回
- 没想到中午下山反而更冷
- 冻得耳朵都痛了
- 七月的脚也冻得不行，下台阶后脚一跳一跳地抬起来
- 终于理解为什么遇到的下山的人面色都比较凝重
- 下山又冷又没有新鲜风景，一股脑就下去了

## 下山中
- 下山路程还剩一半
- 回去路上有点滑
- 直接上冰爪了
